Инструкция по использованию консольного приложения "Записная книжка".

1. Перед запуском программы убедитесь, что у Вас установлена Java Runtime Environment версии 1.6 или выше.

2. Запустите TaskManager.jar. В Windows для этого необходимо в командной строке из каталога
с jar файлом программы вызвать команду "java -jar TaskManager.jar".

3. Вы увидите меню программы с возможностью выбора команд. Для вызова команды необходимо ввести соответсвующую цифру.
	-1.Показать все контакты
	-2.Найти контакт по имени  
	-3.Ввести новый контакт
	-4.Удалить контакт по имени
	-5 Выйти из программы.

4. При создании контакта, Вам будет предложено ввести имя, электронный адрес и телефон. Имя должно состоять как минимум из одного символа. Имена контактов могут повторяться. Электронный адрес и телефон должны быть введены в правильном формате.

5. После первого сохранения контакта, будет создана директория C:/Notebook и файл notes.xml. В этом файле будет храниться вся введенная Вами информация о контактах. 

6. Поиск контакта осуществляется по имени. Вы можете ввести имя полностью или только его часть, поиск выдаст все подходящие результаты. 

7. Удаление контакта производится также по имени, и если у Вас было создано несколько контактов с одним именем, то будет удален тот, который добавлен раньше.

8. При выходе из программы, информация о ранее введенных контактах сохраняется в файле, Вы можете использовать её при последующих запусках.
